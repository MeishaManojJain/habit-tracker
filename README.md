# GoodHabits

A simple habit tracker made in Java.

## What it does

- Lets you create multiple habits.
- Stores the number of hours spent on each habit.
- Shows the hours for all habits.
- Uses a simple menu in the terminal.

## Built with

- Java
- IntelliJ IDEA
- Git

## Running the project

Run `Main.java`.
